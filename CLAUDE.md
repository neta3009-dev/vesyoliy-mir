Создай развивающую веб-игру для детей 2–4 лет "Весёлый мир".

## Технический стек
- Чистый HTML5 + CSS3 + Vanilla JS (без фреймворков)
- Один файл index.html + папка assets/
- Работает без интернета (все ресурсы локальные)
- Адаптив: работает на планшете и телефоне

## Структура: 4 мини-игры с навигацией через большие иконки

### 1. 🐮 Животные
- 8 животных: кошка, собака, корова, лягушка, утка, лошадь, овца, свинья
- Большая карточка с рисунком (SVG или emoji крупно)
- При нажатии: анимация "прыжок" + звук через Web Speech API
- Голос произносит: название животного + его звук ("Корова говорит МУ!")

### 2. 🔴 Цвета и формы
- 6 фигур: круг, квадрат, треугольник, звезда, сердце, прямоугольник
- Каждая в своём цвете (красный, синий, жёлтый, зелёный, оранжевый, фиолетовый)
- При нажатии: фигура увеличивается + голос называет цвет и форму
- Кнопка "Найди красный круг!" — простое задание

### 3. 🔢 Цифры и счёт
- Цифры от 1 до 10
- При нажатии на цифру N — появляется N весёлых предметов (звёздочки/яблоки)
- Голос считает по одному: "Один... два... три!"
- Предметы появляются с анимацией по очереди

### 4. 🔤 Буквы
- Русский алфавит, по 6 букв на экране (можно листать)
- При нажатии: буква увеличивается + голос говорит букву + слово
- Пример: "А — Арбуз", "Б — Барабан", "В — Волк"
- Картинка-иллюстрация рядом (emoji)

## UX-требования (критично для возраста 2–4 года)
- Кнопки минимум 120x120px
- Шрифт минимум 32px
- Цвета яркие, насыщенные — никакого пастельного
- Мгновенный отклик на касание (no delay)
- При любом правильном действии — конфетти (CSS-анимация) + звук похвалы
- Никаких проигрышей, таймеров, счёта очков
- Кнопка "назад" всегда видна и большая
- touch-action: manipulation на всех кнопках

## Голос (Web Speech API)
- Язык: ru-RU
- Скорость (rate): 0.8 — медленно и чётко
- Тональность (pitch): 1.2 — чуть выше обычного, дружелюбно
- Перед озвучкой отменять предыдущую речь (speechSynthesis.cancel())
